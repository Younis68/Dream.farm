# Dream Farm - حجوزات المزارع

هذه نسخة محسّنة من صفحة `index.html` لعرض المزارع، تحتوي على: 

- شريط فلتر (بحث وموقع) قابل لتوليد خيارات الموقع آليًا.
- بطاقات مزارع مع فيديو، خصائص، وسعر.
- نوافذ منبثقة (Modal) للحجز وعرض معرض الصور.
- ربط الحجز بواتسآب عبر رابط مُنسق.
- دعم الوصول (ARIA), إغلاق بواسطة مفتاح Escape، وحبس التركيز داخل المودال.
- تحسينات CSS للشكل والاستجابة (Responsive).

---

## تركيب الملفات 

- `index.html` — صفحة HTML الرئيسية تحتوي على الكل: HTML, CSS وJS.
- `header-bg.jpg`, `wena1.jpeg`, `wena2.jpg`, ... — مفقودات الصور والفيديوهات: ضع ملفاتك في نفس المجلد.

## تعليمات الاستخدام

1. ضع الصور والفيديوهات (`wena1.jpeg`, `gohdar1.mp4`, إلخ) في نفس المجلد بجانب `index.html`.
2. افتح `index.html` في المتصفح (يفضل Chrome/Edge/Firefox) للتجربة.
3. لتعديل مزرعة جديدة، انسخ `<article class="farm-card">` وأضف الصفات:
   - `data-farm-name` — اسم المزرعة
   - `data-farm-price` — السعر الرقمي (بدون فواصل)
   - `data-whatsapp` — رقم واتسآب (بدون +)
   - `data-location-text` — نص الموقع
   - `data-gallery` — قائمة صور مفصولة بفاصلة

4. استخدام شريط البحث لتصفية النتائج حسب الاسم، الخصائص أو المبلغ.
5. اضغط على "احجز الآن" لفتح نافذة الحجز وتعبئة البيانات لإرسال رسالة عبر واتسآب.

## ملاحظات تطويرية

- يمكنك إضافة ترتيب وفرز (Sort) أو فلترة متقدمة (نوع، عدد الغرف، إمكانية الحيوانات الأليفة...)
- يمكن ترقية استدعاء فيديوهات للحفاظ على الاستهلاك من خلال `preload="none"` على الأجهزة الصغيرة.
- تحسين تجربة الوصول: إضافة `aria-live` إلى رسائل الخطأ أو إشعار عند عدم وجود نتائج.
